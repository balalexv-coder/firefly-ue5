# Firefly UE5

Интерактивная одномиссионная сцена во вселенной сериала **Firefly**, на Unreal Engine 5.7.4.
Игрок летит на борту «Serenity», общается с экипажем через диалоги, генерируемые LLM, наблюдает выход корабля на орбиту пустынной планеты на экране в кают-компании, а затем смотрит синематик с приземлением у ковбойского городка, в котором и разворачивается миссия.

> Персонажи и сеттинг — оммаж сериалу *Firefly* (2002). Это некоммерческий fan-проект/обучающий эксперимент с UE5, без притязаний на права Fox/Mutant Enemy/Joss Whedon.

---

## Что это за проект

Одна короткая «вертикаль»:

1. **Сцена 1 — Кают-компания.** Вся команда (9 персонажей) сидит за столом. Игрок выбирает реплику → LLM генерирует 2–3 ответа других членов экипажа → разговор возвращается к игроку. Повторяется несколько раундов. На мониторе в стене — живое изображение корабля, подходящего к планете.
2. **Сцена 2 — Синематик приземления.** Когда подлёт к атмосфере достигает триггера, запускается синематик в стиле сериала: «Serenity» заходит по пологой дуге, поворачивает двигатели в режим висения, поднимает облако пыли на посадке.
3. **Сцена 3 — Ковбойский городок.** Игрок управляет персонажем на поверхности планеты рядом с детализированным западным городком: NPC, лошади, звуковая атмосфера, простая завязка миссии.

## Текущий статус

Репозиторий — это **план и инфраструктура**, не готовый `.uproject`.
Здесь лежат:
- детализированные документы — по каждой подсистеме,
- рабочий **dialogue server** (FastAPI + Claude) для генерации реплик,
- шаблоны ассет-структуры, чек-листы, ссылки на плагины и паки.

Сама UE-разработка ведётся локально в `UnrealProject/` (игнорируется git'ом кроме метаданных).

## Быстрый старт

1. **Клонируй** репо: `git clone git@github.com:balalexv-coder/firefly-ue5.git`
2. **Прочитай** [`docs/PLAN.md`](docs/PLAN.md) — там общий маршрут.
3. **Подними dialogue server** (инструкции в [`tools/dialogue_server/README.md`](tools/dialogue_server/README.md)).
4. **Создай UE-проект** по [`docs/UE_SETUP.md`](docs/UE_SETUP.md) → `UnrealProject/`.
5. Двигайся по фазам `PLAN.md`.
