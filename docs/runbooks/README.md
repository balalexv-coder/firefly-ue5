# Runbooks — пошаговые инструкции с подводными камнями

Эта папка — **архив пройденных воркфлоу**. Каждый файл описывает конкретную задачу: порядок действий, точные названия пунктов меню в UE 5.7, известные ошибки и обходные пути.

## Почему существует

Эпик не поддерживает полной per-option документации UE; многие меню переименовываются между minor-версиями; туториалы в интернете часто описывают UE 5.2-5.4, а мы на 5.7. При повторении задачи (например, ретаргет на ещё одного персонажа) без runbook'а пришлось бы заново гуглить и наступать на те же грабли.

Правило: **если задача заняла больше 30 минут из-за неочевидных UI-путей — пиши runbook, пока память свежая.**

## Как структурировать новый runbook

- Имя файла: `NN_short_name.md` (например `02_metahuman_creator.md`).
- Структура:
  1. **Цель** — что хотим получить на выходе.
  2. **Пререквизиты** — что должно быть готово до старта.
  3. **Шаги** — пронумерованные действия с точными названиями пунктов меню для UE 5.7.
  4. **Подводные камни** — все ошибки/задержки, с которыми столкнулись, и как их обошли.
  5. **Источники** — ссылки на доки и форумы, которые пригодились.
